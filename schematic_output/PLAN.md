# 多智能体协同的空间感知图像恢复方法 - 原理图规划

## 图概述

本图展示"多智能体协同的空间感知图像恢复方法"的完整研究框架，呈现从输入到输出的数据流与三大核心模块的层级关系。

## 布局策略

采用**从左到右的水平流程布局**，符合技术路线图阅读习惯：

```
[输入] → [空间感知模块] → [层级规划模块] → [轻量部署模块] → [输出]
```

## 模块划分

### Group 1: 输入层
- 节点：退化图像（含异质退化：噪声、模糊、雾霾、雨线）

### Group 2: 空间感知模块
- 节点1: SAM/RobustSAM语义分割
- 节点2: VLM退化分析（DepictQA/Q-Instruct微调，链式思维推理）
- 节点3: 空间退化图 M={(R_i, d_i, s_i)}

### Group 3: 层级规划模块
- 节点1: 全局调度器（LLM，基于退化先验排序制定策略）
- 节点2: 区域专家（多个并行专家）
- 节点3: 工具库（去噪器、去模糊器、去雾器、去雨器、超分辨率）
- 节点4: 区域恢复 + 质量反馈闭环
- 节点5: 掩码融合（合成最终恢复图像）

### Group 4: 轻量部署模块
- 节点1: 轨迹收集（离线收集多智能体决策轨迹）
- 节点2: 轨迹蒸馏
- 节点3: 轻量路由网络 Φ_ω（编码器-多任务解码器架构）
- 节点4: 四分支头输出（区域掩码、退化类型、工具路由、参数）

### Group 5: 输出层
- 节点：恢复图像 x̂

## 边连接策略

### 主流程（实线）
1. 退化图像 → SAM分割 → VLM分析 → 空间退化图
2. 空间退化图 → 全局调度器 → 区域专家 → 工具库 → 区域恢复
3. 区域恢复 → 掩码融合 → 恢复图像
4. 轨迹收集 → 轨迹蒸馏 → 轻量路由网络

### 辅助连接（虚线）
- 全局调度器 ↔ 区域专家（双向，指令下发与结果上报）
- 区域恢复 → 质量评估 → 区域专家（反馈闭环，虚线标注"迭代优化"）
- 空间退化图 → 轨迹收集（数据流向，虚线标注"离线轨迹"）
- 轻量路由网络 → 恢复图像（部署阶段推理路径，虚线标注"实时推理"）

## 视觉设计

- 输入/输出节点：使用强调样式（primary）
- 核心处理节点：使用关键样式（critical）
- 辅助工具节点：使用次要样式（secondary）
- 模块分组用不同背景色区分：
  - 空间感知模块：蓝色系
  - 层级规划模块：橙色系
  - 轻量部署模块：绿色系
