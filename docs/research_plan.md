# 研究主题细化方案

## 研究题目

**多智能体协同的空间感知图像恢复方法研究**

英文：Spatially-Aware Image Restoration via Multi-Agent Collaboration

## 现有工作的核心局限（突破口）

通过分析 AgenticIR (ICLR 2025)、MAIR、Q-Agent、RestoreAgent 四篇智能体图像恢复论文，发现它们共享三个关键盲区：

| 局限 | 说明 |
|------|------|
| **全局处理** | 所有方法对整张图像统一处理，忽略了真实退化的空间不均匀性 |
| **工具黑盒调用** | 仅做"用不用"的二元决策，不能调节工具的连续参数（如去噪强度） |
| **推理代价过高** | AgenticIR 单张图需200秒+20次LLM调用；MAIR仍需35秒，远不能实用 |

## 研究内容框架（三个核心模块）

### 模块一：空间感知退化分析（Spatial-Aware Degradation Perception）

- 现有方法（AgenticIR、MAIR、Q-Agent）只做全图级别的退化识别，如"图像有噪声和模糊"
- **创新点**：引入 SAM（Segment Anything Model）进行零样本区域分割，结合微调的 VLM（如 DepictQA 或 Qwen2-VL）对每个语义区域独立做退化分析
- 输出从"全图退化描述"升级为"空间退化图"（Spatial Degradation Map）：哪些区域有什么退化、什么程度
- **新颖性**：目前没有任何智能体方法实现了区域级别的退化感知与修复

### 模块二：层级规划与细粒度工具调度（Hierarchical Planning with Fine-Grained Tool Control）

- 设计两层智能体架构：
  - **全局调度器（Global Scheduler）**：基于空间退化图，决定整体修复策略和区域优先级，借鉴MAIR的退化先验（场景→成像→压缩）来约束搜索空间
  - **区域专家（Region Expert）**：对每个退化区域独立规划，不仅选择工具，还**调节工具参数**（如去噪sigma、超分辨倍率、对比度增强系数），基于质量反馈的贪心策略迭代优化
- **新颖性**：现有方法要么是单智能体DFS搜索（AgenticIR），要么是多智能体但仍全局处理（MAIR）。本方案首次实现了"空间维度上的层级规划"+"连续参数调节"

### 模块三：轨迹蒸馏的轻量路由网络（Trajectory-Distilled Lightweight Router）

- 利用上述系统在大量合成退化图像上离线运行，收集"输入图像特征→最优修复路径+参数"的配对数据
- 训练一个轻量级 CNN/ViT 路由网络，直接根据输入图像预测：区域分割、各区域退化类型、工具选择及参数
- 部署时完全不依赖 LLM/VLM，实现端侧实时推理
- **新颖性**：将智能体系统作为"数据生成器"，用知识蒸馏的思路把复杂推理压缩为轻量网络，目前文献中无人尝试

## 与现有方法的对比

| 对比维度 | AgenticIR | MAIR | Q-Agent | **本方案** |
|---------|-----------|------|---------|-------------|
| 退化分析粒度 | 全图 | 全图 | 全图 | **区域级** |
| 工具控制方式 | 二元选择 | 二元选择 | 二元选择 | **连续参数调节** |
| 规划架构 | 单智能体DFS | 多智能体+先验 | 贪心 | **层级空间规划** |
| 部署要求 | GPT-4 + 双3090 | GPT-4o | 本地7B模型 | **轻量CNN路由** |
| 推理速度 | ~200s | ~35s | ~20s | **目标<1s（蒸馏后）** |

## 技术路线图

```
阶段一 → 构建空间退化感知模块（SAM + VLM 区域退化分析）
阶段二 → 设计层级智能体（全局调度 + 区域专家 + 参数调节）
阶段三 → 在合成 & 真实数据集上验证完整系统
阶段四 → 轨迹收集 + 轻量路由网络蒸馏 + 端侧部署验证
```
